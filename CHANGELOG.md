# Changelog

All notable changes to the claude-gemini-bridge project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Multi-AI Orchestrator agent with 5 collaboration patterns
- AI Compare skill for structured response comparison
- Comprehensive testing checklist
- Task improvement guide generated by multi-AI collaboration
- Configuration hierarchy explanation in INSTALL.md
- WebSearch vs Gemini tool usage clarification

### Changed
- Updated README configuration instructions (mcp.json instead of settings.json)
- Improved error messages with friendly Chinese prompts
- Enhanced JSDoc documentation throughout codebase
- **BREAKING**: Removed model parameter from examples - users should not specify model to avoid "Requested entity was not found" errors

### Fixed
- Removed hardcoded proxy configuration - now supports environment variables
- Added startup health check for Gemini CLI
- Fixed misleading configuration documentation
- Removed redundant .mcp.json file
- Fixed incorrect model parameter examples that caused API errors
- Clarified that MCP Server should be configured at user level (~/.claude/mcp.json)

### Security
- Improved error handling to prevent shell injection
- Better validation of user inputs

## [1.0.1] - 2026-01-07

### Added
- Health check on server startup
- Environment variable support for proxy configuration
- Environment variable support for timeout configuration
- Comprehensive test suite
- Detailed JSDoc documentation

### Changed
- Updated README with correct mcp.json configuration path
- Improved error messages with helpful troubleshooting steps
- Version bumped to 1.0.1

### Fixed
- Removed hardcoded proxy (http://127.0.0.1:7890)
- Fixed documentation inconsistencies
- Better error handling for missing Gemini CLI
- Timeout errors now provide actionable suggestions

### Removed
- Redundant project-level .mcp.json configuration file

## [1.0.0] - 2026-01-07

### Added
- Initial release of Gemini MCP Server
- Three MCP tools: gemini_chat, gemini_compare, gemini_model_info
- AI Compare skill for comparing Claude and Gemini responses
- Multi-AI Orchestrator agent with 5 collaboration patterns
- Comprehensive documentation and examples
- Quick start guide
- Testing checklist

### Features
- Seamless integration between Claude Code and Gemini AI
- Support for multiple Gemini models
- Configurable proxy settings
- Multi-AI collaboration patterns (Sequential, Parallel, Competitive, Collaborative, Supervisor-Worker)
- Global configuration for Skills and Agents

### Documentation
- README.md with project overview
- QUICKSTART.md for 5-minute setup
- TESTING_CHECKLIST.md for verification
- Multiple usage examples
- API reference for MCP tools

- **Patch version (1.0.X)**: Bug fixes, documentation updates
